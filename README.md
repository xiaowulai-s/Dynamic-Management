## 功能特性

### 账号与权限
- **双预设账号**: root(root123) 超级管理员 + admin(admin123) 管理员，另含 admin_test/user_test 测试账号
- **三级角色**: 超级管理员（不可删除）> 管理员 > 用户
- **登录安全**: `【未实现】` 5次输错渐进式锁定（10s→30s→60s→120s→300s），实时倒计时。当前登录失败仅返回 401，无锁定机制
- **用户管理**: 管理员创建用户（创建时手动指定密码，非默认 123456）、通过编辑用户修改密码、角色编辑。`【未实现】` 独立的密码重置功能
- **用户注册**: 支持用户自主注册账号（`/register` 页面）

### 设备管理
- **类型体系**: 设备名称即类型，同类设备通过编号区分
- **组合唯一**: 名称+编号唯一约束，不同类型可用相同编号
- **全字段管理**: 型号、规格、厂家、供应商、安装位置、购置日期、质保期

### 日志管理
- **8 类日志**: 安装、维修、报废、巡检、保养、故障报修、配件更换、校准
- **级联选择**: 提交日志时先选设备类型→再选具体编号
- **日志时间线**: 按设备查看彩色时间线，全部动态字段直接展开
- **审批流**: 待审批日志在详情弹窗中通过/驳回，驳回需填写原因

### 统计分析
- **故障率趋势**: 按时段（天/周/月）统计设备故障率
- **成本分析**: 维修费用统计、平均成本、趋势分析
- **导出功能**: 支持 Excel、PDF、CSV 格式报表导出

### 其他功能
- **用户反馈**: 用户提交问题→管理员回复
- **客户管理**: 客户 CRUD + 设备数量统计
- **操作审计**: 审计日志查看页（`【部分实现】` 仅审计 API 内部记录，登录/增删改等操作尚未接入审计记录）
- **数据备份**: JSON 全量数据导出
- **批量上传与 OCR**: 拖拽上传多文件（最多 20 个），自动触发 OCR 识别，结果可勾选导入（`【部分实现】` 当前 OCR 为模拟实现，easyocr 未安装时降级）
- **通知中心**: 独立通知页 + 顶栏通知气泡，含未读数徽标
- **站点标题动态配置**: 管理员可在系统设置修改中英文名称，实时同步
- **深色模式**: 支持明暗主题切换（侧边栏切换按钮）

## 技术栈

### 后端
- FastAPI (Python 3.11)
- PostgreSQL + SQLAlchemy ORM
- Redis + Celery
- JWT + bcrypt（锁定 bcrypt==4.0.1，兼容 passlib 1.7.4）
- Nginx 反向代理

### 前端
- Vue 3 + TypeScript + Vite
- Element Plus 组件库
- ECharts 数据可视化
- Pinia 状态管理

### 部署
- Docker + Docker Compose
- PostgreSQL + Redis 独立容器
- 端口映射：前端 9000 / 后端 8000 / 数据库 15432 / Redis 6379

**迁移到其他服务器**：参考 [迁移部署指南](docs/迁移部署指南.md)，使用 `迁移-导出.ps1` / `迁移-导入.ps1` 一键迁移全量数据。

## 快速开始

```bash
git clone https://github.com/xiaowulai-s/Dynamic-Management.git
cd Dynamic-Management
docker-compose up -d --build
```

访问 http://localhost:9000
