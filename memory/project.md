## 项目背景

**项目名称**: 设备信息动态管理系统  
**客户需求**: 工业机械设备全生命周期管理  
**核心痛点**:
- 设备信息分散在不同文档中，难以集中管理
- 无法快速查询设备历史维护记录
- 缺乏数据统计分析，难以发现设备问题趋势
- 多人协作时数据不一致，容易产生信息孤岛

## 技术约束

- **部署方式**: 本地内网部署
- **并发需求**: 20人同时使用，小团队协作
- **文件格式**: 支持 Word/PDF 上传和 OCR 识别
- **访问方式**: 响应式网页，支持PC和移动端

## 技术栈

### 后端
- FastAPI 0.104.1
- SQLAlchemy 2.0.23 (ORM)
- PostgreSQL 15
- Celery 5.3.4 + Redis 5.0.1
- PaddleOCR 2.7.3（中文识别）
- PyMuPDF 1.23.8（PDF处理）
- python-docx 1.1.0（Word处理）

### 前端
- Vue 3.3.9 + TypeScript
- Element Plus 2.4.4
- Pinia 2.1.7（状态管理）
- Vue Router 4.2.5
- Axios 1.6.2
- ECharts 5.4.3

## 核心功能

1. **用户与权限管理**: JWT认证，角色管理（管理员/普通用户）
2. **设备基础信息管理**: 设备档案、状态跟踪、生命周期管理
3. **多类型日志管理**: 8种日志类型（安装/维修/报废/巡检/保养/报修/配件/校准）
4. **灵活审批流程**: 管理员可配置哪些日志类型需要审批
5. **文件上传与OCR识别**: 批量上传，自动识别文档内容
6. **统计报表与数据分析**: 故障率、维修成本、保养计划等多维度报表
7. **智能预警提醒**: 定期保养提醒、设备寿命预警、校准到期提醒
8. **数据导出**: Excel和PDF报表导出
