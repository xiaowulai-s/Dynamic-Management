# Changelog

All notable changes to this project will be documented in this file.

## [1.1.1] - 2026-07-16

### Added
- **账号体系**: root(root123) 超级管理员 + admin(admin123) 管理员双预设账号
- **登录锁定**: 5次输错渐进式锁定（10s→30s→60s→120s→300s），实时倒计时禁用输入
- **密码安全**: 6-20位密码，修改后自动跳转登录页，错误提示改为内联警告框
- **角色权限**: 超级管理员不可删除，仅超管可删管理员，admin+super_admin均可审批
- **用户昵称**: 用户表新增 nickname 字段，个人信息页可编辑
- **用户头像**: 新增 avatar_url 字段，支持自定义头像 URL，Header 优先显示图片
- **设备类型体系**: 设备名称即类型，编号+名称组合唯一约束，不同类型可用相同编号
- **设备类型筛选**: 搜索栏新增设备类型下拉
- **日志级联选择**: 提交日志时先选设备类型→再选具体编号
- **设备日志时间线**: 新建 /equipment/:id/logs 页，彩色时间线全字段展开，含时间范围筛选
- **日志编号列**: 日志管理表新增设备编号列
- **日志审批流重构**: 审批集成到详情弹窗，列表仅显示状态标签，驳回需填原因
- **仪表盘卡片跳转**: 四个统计卡片全部可点击跳转对应页面
- **状态文字列表**: 设备状态饼图旁新增各状态数量文字展示
- **日志类型统计**: 从 API 实时拉取真实日志类型数量
- **用户反馈**: 全新模块，用户提交反馈+管理员回复，状态标签
- **客户管理**: 全新模块，客户 CRUD + 关联设备数统计
- **操作审计**: 全新模块，记录用户操作日志，仅管理员可查看
- **数据备份**: JSON 全量数据备份导出接口
- **系统设置初始化**: 审批配置 8 项 + 系统配置 6 项默认值
- **系统名称同步**: 修改系统名称后左上角 Logo 旁文字实时更新
- **通知中心**: 重写为 Naive UI，未读高亮 + 点击已读

### Changed
- **UI 组件迁移**: Login/Profile/LogDetail/Notifications 全部从 Element Plus 迁移至 Naive UI
- **深色模式**: 切换按钮注释隐藏（CSS 变量保留），默认浅色模式
- **注册入口**: /register 路由注释，注册链接移除（用户由管理员创建）
- **设备编辑规则**: 编号可修改，类型默认只读（管理员可改）
- **用户管理页**: 三表折叠面板（超管/管理/用户），A-Z隐性排序
- **个人信息入口**: 从侧边菜单移至右上角下拉菜单
- **统计分析图表**: 所有 ECharts 图表 bottom 间距从 3%→12%，X轴标签旋转30度
- **统计分析导出**: 全部 Excel/PDF 导出端点验证可用
- **操作员默认值**: 提交日志时操作人默认为当前登录用户

### Fixed
- **审批权限**: super_admin 无法审批日志问题修复
- **设置切换失败**: 审批配置开关 update 失败（模型字段改为 Optional）
- **系统配置更新**: 系统参数编辑无效问题修复
- **设置 token**: Settings.vue 使用 userStore.token 替代 localStorage
- **设备编号同步**: 日志列表通过 JOIN 实时获取最新设备编号
- **故障率图表**: 文字遮挡修复（底部间距+标签旋转）
- **反馈入口**: 侧边菜单补充用户反馈入口
- **Profile 表单**: 缺失昵称 form-item 标签修复
- **后端语法错误**: sed 批量替换导致的 avatar_url 和 role 重复参数修复
- **审计模型冲突**: 移除重复的 AuditLog 模型定义

## [1.0.0] - 2026-07-13

### Added
- 基于 FastAPI + Vue 3 的设备全生命周期管理系统初始版本
- 8 类日志管理（安装/维修/报废/巡检/保养/故障/配件/校准）
- 文件 OCR 识别批量上传
- 设备管理与质保预警
- 仪表盘统计分析
- Element Plus UI 框架
- Docker Compose 容器化部署
- 腾讯云部署指南
